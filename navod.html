Time data in documents — Guide

The aim of this project is to create a script which

  • displays a timeline with all time stamps and their discriptions in the controls part of an HTML document
  • displays controls changing the format of displayed timestamps between absolute and relative

Usage:

  • include the script using sequnce: <script src = "timestamps.js" type = "text/javascript"/></script>
  • include appropriate CSS style using sequnce: <link href = 'style-timestamps.css' rel = 'stylesheet' type = 'text/css'/>
  • add an empty div class="controls" element to HTML document to the place, where the timeline should be displayed
  • create and show a timeline by calling main, for example by adding: <body onload = "main()">
An example of the script usage can be found in demo.html.
